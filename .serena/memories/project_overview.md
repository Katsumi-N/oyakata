# プロジェクト概要

## プロジェクトの目的
製図試験の勉強用プリント管理iOSアプリ。建築士試験の製図勉強において、問題用紙、エスキス、製図などの画像をアップロード・管理し、Apple Pencilで編集・注釈を追加できるアプリです。

## 技術スタック
- **プログラミング言語**: Swift
- **UIフレームワーク**: SwiftUI
- **データ永続化**: SwiftData
- **アーキテクチャ**: MVVM パターン
- **ターゲットプラットフォーム**: iOS
- **テストフレームワーク**: XCTest
- **対応デバイス**: iPhone/iPad（Apple Pencil対応）

## 主要機能
- 画像のアップロード・編集（Apple Pencil対応）
- 4つの定義済みタグによる分類：
  - 問題用紙
  - エスキス
  - 製図
  - 解答例
- タイトルによる画像管理（必須項目）
- ミスリスト機能（製図後の振り返り用）
- 検索・フィルタリング機能
- ローカルストレージ（画像はデバイス内に保存）
- 画像のグループ化機能
- 時間記録機能

## 非機能要件
- ローカル保存（クラウドストレージは使用しない）
- モダンでユーザーフレンドリーなデザイン
- レスポンシブ対応（iPhone/iPad両対応）
